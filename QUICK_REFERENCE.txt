🔧 COMMON TASKS
─────────────────────────────────────────────────────────────────────────────

DELETE A USER:
  1. Go to /superadmin/database/
  2. Select "Users" from dropdown
  3. Find user in list
  4. Click "Delete" button
  5. Confirm action
  ✓ Action automatically logged in audit trail

VIEW ALL STUDENTS:
  1. Go to /superadmin/database/
  2. Select "Students" from dropdown
  3. Browse 50 per page (use pagination)
  4. View details: name, email, branch, status

EXPORT DATABASE:
  1. Go to /superadmin/exports/
  2. Choose format (CSV/Excel/JSON/PDF)
  3. Select tables to include
  4. Click "Create Export"
  5. Download when ready

CREATE BACKUP:
  1. Go to /superadmin/backups/
  2. Click "Create Backup Now"
  3. Select type (Full Database)
  4. Wait for completion
  5. Download and store safely

CHECK AUDIT LOG:
  1. Go to /superadmin/audit-log/
  2. Filter by action/user/status (optional)
  3. View all SuperAdmin activities
  4. See IP addresses and timestamps

ADD ANOTHER SUPERADMIN:
  $ python manage.py shell

  from django.contrib.auth import get_user_model
  from superadmin.models import SuperAdmin

  User = get_user_model()
  user = User.objects.create_user(
      email='admin2@dristisewa.com',
      password='SecurePassword!',
      is_staff=True,
      is_superuser=True
  )
  SuperAdmin.objects.create(user=user)
